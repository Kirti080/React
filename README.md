# EmpFlow Employee Management

A React + Vite starter for an IT company employee management dashboard.

## Features

- Workforce KPI cards for headcount, utilization, hiring, and tenure
- Employee directory table with status and allocation details
- Team capacity overview
- HR pending-action queue
- Responsive layout for desktop and mobile screens

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Development

Run the local dev server:

```bash
npm run dev
```

Then open the local Vite URL shown in the terminal.
